░ HTML Tags

abbr
An acronym or abbreviation, like “HTML”, “CSS”, etc. title attribute contains the expanded version, like "HTML", "CSS"
blockquote
Specifies a section that is quoted from another source.
cite
A citation for another source, often used with quotations. A person’s name, a URL, a book, a movie title, etc.
div
A division or a section in an HTML document. Any sort of content can be put inside the div tag!
em
Used to define emphasized text. The content inside is typically displayed in italic. A screen reader will pronounce the words in em with an emphasis, using verbal stress.
